# Design Prompt — CHROMZY (send to Claude / design tool)

Copy everything below the line into the design tool.

---

Design a **light, editorial, gallery-grade** landing site for **CHROMZY**, a generative digital-art collection of 4,444 works on Solana. The brief is to look **deliberately hand-designed by a human art director — the opposite of a generic dark neon NFT mint page.**

**Hard rules (these kill the generic look):**
- **Light theme.** Warm off-white / paper background (#F4F1EA-ish), ink-black text (#16140F). NOT dark mode. NO neon.
- **No** glitch effects, scanlines, film grain, holographic cards, orbiting rings, terminal/console feeds, scrolling trait marquees, or rarity % bars. These are banned — every competitor uses them.
- **No** mystical "4,444 souls / the chain dreamed" lore voice. Copy is plain, confident, specific.
- One restrained ink accent only (a single saturated hue — e.g. cobalt #1E3AFF or vermilion #E8421A). Used sparingly, like a printer's spot color.

**Aesthetic:** Swiss/editorial print meets contemporary art-museum catalogue. Think a printed art book or a gallery exhibition site, not a crypto product. Reference feeling: Art Blocks' restraint + a printed magazine's confidence.

**Typography:** Large expressive display — a high-contrast serif (e.g. a modern didone) OR a characterful grotesque, paired with a clean neutral sans for body and a mono only for spec/numbers. Type is the hero. Use real typographic scale — huge headlines, tight leading, generous margins.

**Layout:** Asymmetric editorial grid. Generous whitespace. Hairline rules and a visible baseline grid feel. The artwork is framed like numbered plates in a catalogue (e.g. "Plate 0042 / Drifter"), each **attributed to a named artist**. One quiet colophon-style spec table for chain/supply/price instead of a stats bar.

**Signature moment:** ONE memorable interaction, used once — e.g. a single large generative artwork that subtly reacts to cursor/scroll, or a curated plate that re-composes. Not ten competing effects.

**Pages:** Home, Collection (catalogue grid + filters), Mint (focused, progressive wallet connect), About (artist + story), Roadmap, Team, FAQ. Above the fold show chain + price + supply + one CTA before any wallet connect.

**Tone of copy:** Curatorial and direct. Sells the art and the makers, with real receipts (supply, artist names, on-chain facts) — never hype.

Deliver a distinctive hero composition first, then the section system. Avoid centered-everything symmetry, avoid the standard left-text/right-image hero — surprise me with the composition.
